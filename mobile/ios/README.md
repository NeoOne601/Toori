# iOS Project Notes

Open [TooriLens.xcodeproj](/Users/macuser/toori/mobile/ios/TooriLens.xcodeproj) directly in Xcode.

The project is configured as a SwiftUI app target named `TooriLens` and points at the sources in `TooriApp/`.

Recommended first checks in Xcode:

1. Select an iPhone simulator.
2. Set `TOORI_RUNTIME_URL` in the scheme environment if your runtime is not on `http://127.0.0.1:7777`.
3. Build and run.
